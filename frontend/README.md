# OpenOnCall frontend

React 18 + Vite + TypeScript SPA. See the repo root [README](../README.md) for the quickstart.

## Local development (outside Docker)

```bash
npm install
npm run dev      # http://localhost:5173, expects the API at VITE_API_BASE_URL (see .env)
npm run test     # Vitest + React Testing Library
npm run lint      # oxlint
npm run build     # production build to dist/
```
